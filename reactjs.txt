Setup

1.Download NodeJS and VS Code editor.

2. Build app (Reference: https://github.com/facebook/create-react-app)
	Option 1: using npx
>npx create-react-app my-app (npx is npm package runner. bundled with node.)
Need to install the following packages:
  create-react-app
Ok to proceed? (y)
>cd my-app
>npm start

	Option 2: using npm
>npm install create-react-app -g
>create-react-app <projectname>	
----------
Code snippet
1. rfce <tab> for skeleton with function
2. rce <tab> for skeleton with class
3. rconst <tab> for constructor
----------
Four approaches for event binding
1. Binding in render -                   this.<handler>.bind(this)
2. Binding in render using '=>' -        {() => this.<handler>}
3. Binding in class constructor -        this.<handler> = this.<handler>.bind(this)
